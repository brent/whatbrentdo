NBA Today is a small app I built to display a brief and attractive rundown of today's NBA schedule. I built it to help me answer two simple questions: which teams are playing today and which team might win their matchup?

The app data came from a third party API that became unreliable so the data is no longer fresh. You can still see data from the last day captured, however.
